this program will build a standing table to a soccer league.

the team class is responsible for updating the needed information for each team
    States
        name
        abbreviation
        points
        win
        tie
        loss
        goalScored
        goalAgainst
        goalDifference
    Behavior
        Update points
        Update win
        Update loss
        update tie
        Update goalScored
        Update goalAgainst
        Update goalDifference

game class
the game class will take team and calculate which team won by how many goals.
